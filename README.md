# JDBC Practice

Java Database Connectivity (JDBC) practice files while learning backend development.

## Topics Covered
- Database Connection
- Table Creation using Java
- Data Insertion (Static & Dynamic)
- Data Updation
- Data Selection
- ResultSet

## Tech Stack
- Java
- JDBC
- MySQL

## Prerequisites
- Java JDK installed
- MySQL installed
- MySQL Connector JAR — download from:
  https://dev.mysql.com/downloads/connector/j/

## Setup
1. MySQL Connector JAR download karo
2. Same folder mein rakho
3. Database banao MySQL mein

## How to Run
**Compile:**
javac -cp ".;mysql-connector-j-9.7.0.jar" FileName.java
**Run:**
java -cp ".;mysql-connector-j-9.7.0.jar" FileName

## Files
| File | Description |
|------|-------------|
| jdbcconnection.java | Database connection |
| jdbctablecreation.java | Table create karna |
| jdbctableinsertion.java | Static data insert |
| jdbcdynamicinsertion.java | Dynamic data insert |
| jdbcupdate.java | Data update karna |
| jdbcselect.java | Data select karna |
